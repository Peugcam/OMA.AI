## 📱 Como Usar (Super Simples)

### 1. Enviar Produto para o Leão

Abra o WhatsApp e envie para o bot:

```
📸 FOTO DO PRODUTO
+
📝 Informações:

Nome: [Nome do Produto]
Descrição: [Breve descrição]
Preço: [Preço ou "Consulte"]
Link: [Seu link de afiliado]
Categoria: [beleza/perfumes/electronics/feminine_hygiene]
```

**Exemplo Real:**

```
📸 [Foto do sérum anti-idade]

Nome: Sérum Vitamina C Premium
Descrição: Sérum anti-idade com 20% de vitamina C pura, reduz manchas e linhas de expressão em 7 dias
Preço: R$ 89,90
Link: https://amzn.to/3xYzAbc
Categoria: skincare
Público: mulheres 25-45 anos
Diferenciais:
- Dermatologicamente testado
- Resultados em 7 dias
- Fórmula importada
```

### 2. Aguardar Processamento

O Leão vai:
1. ✅ Analisar a imagem do produto
2. 🔍 Pesquisar melhores abordagens de venda
3. ✍️ Criar copy persuasivo otimizado
4. 🎬 Gerar vídeo profissional com avatar
5. 📤 Enviar anúncio pronto

**Tempo:** 30-60 segundos

### 3. Receber Anúncio Pronto

Você receberá:
- 🎥 Vídeo de 30s pronto para postar
- 📝 Script usado
- 💡 Insights de mercado utilizados
- 💰 Custo: ~$0.12-0.20

## 💡 Dicas para Maximizar Conversões

### ✅ Informações Essenciais

Sempre forneça:
1. **Foto de alta qualidade** - Produto bem iluminado
2. **Nome exato** - Como aparece na loja
3. **Preço** - Ou "Consulte" se variar
4. **Link de afiliado** - Seu link rastreável
5. **Categoria** - Para estratégia otimizada

### ✅ Informações Opcionais (Melhora Resultado)

```
Público-alvo: [quem mais compra]
Diferenciais: [o que torna único]
Garantia: [se tiver, mencione]
Frete: [grátis? mencione!]
Oferta especial: [desconto atual]
```

### ✅ Melhores Práticas

1. **Use fotos do produto em uso** (quando possível)
2. **Mencione benefícios, não features**
   - ❌ "Tem vitamina C"
   - ✅ "Reduz manchas em 7 dias"

3. **Inclua prova social real**
   - Avaliações da Amazon/Mercado Livre
   - Número de vendas
   - Aprovações profissionais

4. **Crie urgência genuína**
   - Oferta por tempo limitado
   - Estoque baixo (se verdade)
   - Bônus que expira

## ❓ FAQ

**P: Posso usar qualquer link de afiliado?**
R: Sim! Amazon, Mercado Livre, Hotmart, Monetizze, Eduzz, etc.

**P: O vídeo já sai com o link?**
R: O link é mencionado no áudio ("link na descrição"). Você adiciona o link na legenda ao postar.

**P: Quantos anúncios posso criar?**
R: Ilimitados! Pague apenas pelos créditos de API usados.

**P: Posso editar o anúncio depois?**
R: Você recebe o script completo. Pode solicitar ajustes ou gerar nova versão.

**P: Funciona para produtos digitais?**
R: Atualmente otimizado para produtos físicos. Produtos digitais em breve!
